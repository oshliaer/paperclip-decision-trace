# CLAUDE.md — Decision Trace Plugin

## Правила изоляции (обязательные)

### Абсолютные пути — запрещены

Никогда не коммитить абсолютные пути файловой системы оператора:

- Запрещено: `/home/user/...`, `/Users/someone/...`, `C:\Users\...`
- Запрещено: любые пути, начинающиеся с домашней директории конкретного пользователя
- Допустимо: `./relative/path`, `$PLUGIN_DIR`, переменные окружения

### Зависимости

- Зависимости устанавливаются только через npm/registry
- Запрещено использовать `file:../local/path` в `package.json` для production-зависимостей
- `.paperclip-sdk/` — исключение для локальной разработки с пакованными тарболами SDK (в .gitignore)

### Paperclip API

- URL Paperclip API передаётся через переменную окружения `PAPERCLIP_API_URL`
- Значение по умолчанию `http://127.0.0.1:3100` допустимо как fallback в коде
- Ссылки на `localhost` или `127.0.0.1` в коде и конфигурации — допустимы

### Секреты и конфигурация

- Файлы `.env`, `.env.local`, `.env.*.local` должны быть в `.gitignore`
- Никогда не коммитить API ключи, токены или приватные конфигурации

## Контекст проекта

Плагин публичный, разрабатывается для интеграции с Paperclip. Лексические пересечения с API и конфигурацией Paperclip допустимы — это штатная интеграция.
